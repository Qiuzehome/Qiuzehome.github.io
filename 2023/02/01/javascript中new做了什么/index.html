2.new做了什么

在new一个实例对象的时候，其实它做了以下操作

  1. 创建一个空对象
  2. 将空对象的原型链指向构造函数的原型链
  3. 执行构造函数，将this绑定到空对象上
  4. 根据构造函数的返回值做判断，若是原始数据则忽略，若是对象的话则正常返回处理
